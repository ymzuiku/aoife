## 常用小方法

Aoife 提供了一些常用方法

### 去抖动 debounce

```jsx
<button debounce="onclick,ontouchstart" onclick={handleClick}>
  频繁点击我
</button>
```

### 节流 throttle

```jsx
<button throttle="onclick" onclick={handleClick}>
  频繁点击我
</button>
```

### 编写 css

```jsx
const css = (
  <style>{`
.hello {
  background: #f00;
}
`}</style>
);

document.body.append(css);
```
